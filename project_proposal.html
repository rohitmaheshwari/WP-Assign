Project Description

Online Tour and Travel Services project is a project for maintaining touring destination information. The objective of this application is to develop a system for automation activities of a travel agency. Tourist can view the preview of destination and get to know about ticket availability and also buy tickets online.

Project Features

  1. User Registration
  2. Hotel Booking
  3. Flights Booking
  4. Tour Plan on Destination
  5. Custom tour plan with hotels and flight availability
  6. Show Travel History
  7. Customer Reviews
  8. Non-registered Customer Feedback
  9. Chatting System between customer and admin
  10. Google Map APIs to show location
  11. Payment Gateway Integration
  12. Reciept Generation
  13. Daily Report Generation
